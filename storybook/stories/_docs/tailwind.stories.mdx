# Tailwind CSS

This design system uses [Tailwind CSS](https://tailwindcss.com/). For more information on how to use Tailwind, check the [Tailwind Docs](https://tailwindcss.com/docs).

## Best Practices

Overall, we aim to use inline classes whenever possible. When not possible, we use extracted classes with tailwind's `@apply` directive.

## Tailwind CSS Plugins

We also use the `@tailwindcss/forms` plugin. View the [docs](https://github.com/tailwindlabs/tailwindcss-forms)
