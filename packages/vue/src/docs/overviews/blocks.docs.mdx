# Block Components

Self contained components that combine multiple other components or are otherwise more complex than base components. Blocks can be easily added to the flow of a page, though some may require additional markup to achieve the desired layout.

- [Hero blocks](#hero-blocks)
- [Text and information blocks](#text-and-information-blocks)
- [Media and media galleries](#media-and-media-galleries)
- [Card types](#card-types)
